CIFAR-10 - Reconocimiento de objetos en imágenes

CIFAR-10 es un conjunto de datos de visión artificial establecido que se utiliza para el reconocimiento de objetos. Los datos CIFAR-10 consisten en 60.000 (32×32) imágenes en color en 10 clases, con 6.000 imágenes por clase. Hay 50.000 imágenes de entrenamiento y 10.000 imágenes de prueba en los datos oficiales. Las clases de etiqueta en el set de datos son:

  • aeroplano
  • automovilístico
  • pájaro
  • gato
  • ciervo
  • can
  • rana
  • ecuestre
  • navío
  • camioneta

El conjunto de datos CIFAR-10 es una colección de imágenes que se utilizan comúnmente para entrenar el aprendizaje automático y los algoritmos de visión artificial. Es uno de los conjuntos de datos más utilizados para la investigación del aprendizaje automático. Dataset

Los datos originales se encuentan expresados en una matriz numpy numérica de (10000 x 3072). El número de columnas (10000) indica el número de datos de la muestra. Como se indica en el conjunto de datos CIFAR-10/CIFAR-100, el vector de fila (3072) representa una imagen en color de 32x32 píxeles. Dado que este proyecto va a utilizar CNN para las tareas de clasificación, el vector de fila original no es apropiado. Para introducir los datos de una imagen en un modelo CNN, la dimensión del tensor de entrada debe ser (anchura x altura x canal_numérico) o (anchura x anchura x altura).
