3.1. Creación de la infraestructura en la nube

Portal Azure

Para evitar la necesidad de hardware específico para el desarrollo de este proyecto, se ha optado por hacer uso de la nube de Azure para crear una máquina virtual. Esta máquina tiene un sistema operativo Linux sobre el que se ha generado un contenedor con Docker.

Contenedor Docker

Gracias a este entorno virtualizado, todo el código del proyecto, parámetros de configuración y las dependencias están totalmente desacopladas, almacenadas en un repositorio en la nube y gestionadas mediante el control de versiones de DockerHub. Este contenedor está accesible desde el siguiente enlace: Docker afrago

Jupyter Notebook colaborativo

Una de las principales ventajas del uso de la computación en la nube a sido el hacer el desarrollo de todo el proyecto accesible desde internet a los diferentes colaboradores siempre y cuando la máquina virtual se encontrase arrancada. Con el uso de Jupyter Notebook y abriendo los puertos correspondientes en la máquina virtual se consigue un entorno colaborativo a la hora de realizar pruebas sobre el código Python. 3.2 Proyecto de clasificación de música

Las pruebas iniciales sobre redes neuronales para la clasificación de música se han realizado con un proyecto disponible en Github; chen0040/keras-audio: keras project for audio deep learning

Keras-audio: Proyecto Keras para aprendizaje profundo de audio.

En el proyecto se utiliza un conjunto de datos para entrenar al clasificador de música para que sea capaz de reconocer el género de las canciones. Este conjunto de datos está alojado en University of Victoria

y consiste en una serie de audios MP3 de diferentes géneros musicales; blues, classical, country, hiphop, jazz, metal, pop, reagge, rock. El clasificador funciona convirtiendo un archivo de audio o de canción en un espectrograma mel que se puede entender como una matriz tridimensional de forma similar a una imagen.
Dentro de las bibliotecas del proyecto keras-audio nos encontramos tres archivos que hace referencia a las diferentes estructuras de redes neuronales con las que se han realizado las pruebas de clasificación de música por género.
